# Análise Empírica de Algoritmos

Projeto desenvolvido com o objetivo de realizar análises empíricas de algoritmos, observando comportamento, desempenho e complexidade através de experimentos práticos e medições de execução.

## Objetivo

O projeto busca comparar algoritmos por meio de testes práticos, permitindo analisar fatores como:

- Tempo de execução
- Crescimento de complexidade
- Eficiência em diferentes entradas
- Comportamento em cenários variados

A proposta é complementar o estudo teórico de análise de algoritmos com observações empíricas obtidas na prática.

## Tecnologias Utilizadas

- Python
- Bibliotecas padrão da linguagem
- Ferramentas para medição de desempenho e análise de resultados
